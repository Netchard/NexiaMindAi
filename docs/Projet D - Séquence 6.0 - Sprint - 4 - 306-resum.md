## 🏗️ **Architecture Implémentée**

### **Pattern de Gestion d'État**
- **ConversationsContext** dans `layout.tsx` pour partager l'état entre toutes les pages `/chat/*`
- Pas de librairie externe (pas React Query, pas Zustand) - conforme aux contraintes
- Cache simple (1 min TTL) pour éviter les recharges inutiles

### **Flux de Données**
```
URL (/chat ou /chat/{id})
    ↓
layout.tsx ( charge conversations + messages )
    ↓
ConversationsContext ( état partagé )
    ↓
[conversationId]/page.tsx ou /chat/page.tsx
    ↓
ConversationList, ConversationHeader, ChatMessageList
```

### **Persistance des Filtres (ST-304)**
- Filtres stockés **par conversationId** dans `conversationStates`
- Nouvelle conversation = filtres par défaut
- Changement de conversation = restauration des filtres de cette conversation
