La paire de spines (DESIGN.md + EXPERIENCE.md) est solide et proche d'être prête pour l'implémentation. Ancrage inhabituellement rigoureux : héritage colors/rounded octet-identique avec le chat, et chaque citation de code (Navbar.tsx:31, chat/page.tsx:48, useAuth.ts, AuthProvider.getSafeRedirect, exports de conversations.ts, PUBLIC_PAGE_ROUTES de proxy.ts) vérifiée exacte. Revue accessibilité : non conforme AA tel qu'initialement rédigé — un échec de contraste critique sur les CTA (texte blanc sur dégradé orange, ~3.0–3.8:1) et un échec sur le libellé au-dessus des puces (~4.08:1), plus plusieurs lacunes comportementales (persistance de tooltip, activation d'un aria-disabled, annonce de navigation SPA, sémantique de titres, mouvement réduit). Toutes ont été résolues dans les spines, sauf le défaut de contraste du dégradé lui-même sur les bulles de message du chat déjà en production — accepté comme dette technique connue par décision produit du 2026-07-18, hors périmètre de cette session.

Accessibility review

review-accessibility.md

Non conforme AA tel qu'initialement rédigé. L'intention était bonne à plusieurs endroits (aria-disabled focusable, tooltip hover+focus, ring de focus visible, boutons sémantiques) mais un échec de contraste dur touchait les deux CTA de conversion, un second touchait un libellé porteur de sens, et plusieurs lacunes comportementales restaient non spécifiées. Toutes corrigées sauf une, actée comme dette technique par décision produit.

critical

Texte blanc sur dégradé CTA sous le seuil AA

DESIGN.md § Colors, § Components (cta-primary/closing-cta)
~3.03:1 à l'extrémité #F4693F, ~3.81:1 à l'extrémité #E64F2B — sous les 4.5:1 requis pour du texte normal (15.5px/600 ne se qualifiait pas "grand texte"). Ce même dégradé+texte blanc est déjà en production sur les bulles de message du chat (ChatMessage.tsx).
Fix: cta-label passé à 19px/700 (qualifie "grand texte", seuil AA 3:1) — corrige les CTA de l'accueil sans toucher au dégradé partagé ni aux bulles de chat déjà en prod.
✓ Corrigé pour l'accueil. Décision produit (2026-07-18) : le même défaut sur les bulles de chat en production est accepté comme dette technique connue, non traité ici (chat déjà status:final, hors périmètre).
